Welcome to my database project!

This project (yet unnamed) is planned to be an open-source and intuitive database for college admissions offices to use for event management, email communications, and application review.

View the project files and documentation at this website

If you have any questions or suggestions, please reach out to me! viplawnboy@gmail.com

Navigating this website

The pages on the left side of the screen each lead to different databases filled with data I've created. Try playing around with the embedded databases!

Connection to Firebase

The current focus of the project is to connect NoSQL databases on Firebase to this website. So far, only the students page displays a database being hosted on Firebase, but the structure from that page will be helpful in figuring out the other categories for the website. The goal of this stage is to make the data in Firebase viewable, queryable, and editable from this website using an intuitive interface.
