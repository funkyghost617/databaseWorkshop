Welcome to my database project!

This project (yet unnamed) is planned to be an open-source and intuitive database for college admissions offices to use for event management, email communications, and application review.

Navigating this website

The pages on the left side of the screen each lead to different databases filled with data I've created. Try playing around with the embedded databases!

Connection to Turso

The current focus of the project is to connect SQLite databases on Turso to this website. I am not yet sure how to do this; however, it seems possible for Turso to host the databases themselves for little to no cost, and those databases can be linked to this website with JavaScript. The goal of this stage is to make the data in Turso viewable, queryable, and editable from this website using an intuitive interface.
